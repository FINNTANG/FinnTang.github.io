# Venom Pulse 项目所需资源文件

## 文件夹结构
请在项目根目录创建以下文件夹和文件：

```
venompulse/
├── cover.png              # 项目卡片封面图（在作品集网格中显示）
├── venompulse-overview.png   # Overview 部分的主图
├── venompulse-process-1.png  # Process 部分的图片
├── venompulse-final-1.png    # Results 部分的第一张图
├── venompulse-final-2.png    # Results 部分的第二张图
└── venompulse-final-3.png    # Results 部分的第三张图
```

## 图片尺寸建议

### cover.png
- **用途**：作品集网格中的项目卡片封面
- **建议尺寸**：1200 x 900 px (4:3 比例)
- **建议内容**：Venom Pulse 玩具枪的主视图或最具代表性的渲染图

### venompulse-overview.png
- **用途**：项目详情页 Overview 部分
- **建议尺寸**：1600 x 1000 px (16:10 比例)
- **建议内容**：完整的产品渲染图，展示整体设计

### venompulse-process-1.png
- **用途**：Development Process 部分
- **建议尺寸**：1400 x 1000 px
- **建议内容**：设计过程图，可以是：
  - 草图/概念稿
  - 线框模型
  - 建模过程截图
  - 设计迭代对比

### venompulse-final-1.png, venompulse-final-2.png, venompulse-final-3.png
- **用途**：Final Outcome 部分的成果展示
- **建议尺寸**：1600 x 1000 px (16:10 比例)
- **建议内容**：
  - final-1: 主渲染图（最佳角度）
  - final-2: 细节特写（展示材质和机械细节）
  - final-3: 不同角度或配色变体

## 图片格式要求
- 格式：PNG（推荐，支持透明背景）或 JPG
- 背景：建议使用纯黑色背景或透明背景，与网站整体风格一致
- 质量：高分辨率，适合网页展示

## 注意事项
1. 所有图片文件名必须与代码中的路径完全匹配
2. 如果暂时没有某些图片，可以先用占位图，稍后替换
3. 建议使用一致的渲染风格和背景色，保持视觉连贯性

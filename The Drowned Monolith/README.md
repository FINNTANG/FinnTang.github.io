# The Drowned Monolith - 视频文件说明

## 📹 如何添加视频

为了在本地正常播放视频，请按以下步骤操作：

### 步骤 1：下载视频
1. 访问 YouTube 视频：https://youtu.be/hqiESOiOgnM
2. 使用在线工具下载视频（推荐：https://www.y2mate.com/ 或 https://yt5s.io/）
3. 下载为 MP4 格式

### 步骤 2：重命名并放置文件
1. 将下载的视频文件重命名为：`drowned-monolith.mp4`
2. 将文件放入此文件夹：`The Drowned Monolith/drowned-monolith.mp4`

### 步骤 3：刷新网页
- 刷新浏览器，视频应该可以正常播放了！

---

## 🌐 在线部署
当你将网站部署到服务器（如 Vercel、Netlify 等）时，YouTube iframe 会自动正常工作，不需要本地视频文件。

## ⚠️ 为什么本地不能用 YouTube 嵌入？
本地文件（file://）协议不支持 YouTube 的 autoplay 和其他权限策略，这是浏览器的安全限制。

---

**视频规格建议：**
- 格式：MP4 (H.264)
- 分辨率：1920x1080 或更高
- 码率：5-10 Mbps
- 音频：AAC

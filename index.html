PHILOSOPHY & APPROACH

My work exists at the intersection of digital innovation and traditional craftsmanship. I believe that the most compelling experiences emerge when computational thinking meets physical making—when algorithms inform woodworking, when AI enhances human connection, and when speculative design addresses real-world challenges.

Whether developing AI-powered virtual pets, crafting sculptural wooden bowls, or designing educational games about ocean conservation, my approach remains consistent: thoughtful problem-solving, iterative experimentation, and deep respect for both material properties and user needs.

Overview

REALITYEATER is a virtual pet browser game where players use real-world items to feed a glitchy, sentient digital pet. It blends AI-powered interactivity with narrative design.

The project explores the intersection of physical and digital reality through webcam-based object recognition, creating an immersive cyberpunk experience.

Development Process

AI Integration & Object Recognition

Built an interactive pet with camera-based object recognition using the OpenAI API, enabling real-world object interaction through the webcam.

Cyberpunk Visual Design

Designed glitch-style UI inspired by The Matrix, using pixel art aesthetics and terminal-like interfaces to create an authentic cyberpunk atmosphere.

Behavioral Logic System

Created JSON-based interaction logic and emotional states for the pet, balancing gameplay through hunger timers and user feedback loops.

Final Outcome

Live prototype available at digital-pet-code-world.vercel.app

Players can interact with the pet through typed messages and feed it real objects via webcam input, creating a unique blend of AI-powered gaming and reality interaction.

Key Learnings

• How to integrate AI APIs into interactive web experiences

• Importance of user feedback in balancing novelty with usability

• Crafting a unique digital personality through minimalist UI and behavior design
